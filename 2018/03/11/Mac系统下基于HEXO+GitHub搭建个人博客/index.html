1. 环境安装

在本地一共要安装 hexo/git/nodejs/homebrew 四个工具；

  • 安装Homebrew，它是一款Mac OS平台下的软件包管理工具，自行百度
  • 安装Git
    $ brew install git
  • 安装nodejs，直接去官网下载安装即可；
  • 安装hexo
    $ sudo npm install hexo-cli -g
    支持本地环境已经搭建完成；

缺失模块。
1、请确保node版本大于6.2
2、在博客根目录（注意不是archer根目录）执行以下命令：
npm i hexo-generator-json-content --save
3、在根目录_config.yml里添加配置： 
jsonContent:
  meta: false
  pages: false
  posts:
    title: true
    date: true
    path: true
    text: false
    raw: false
    content: false
    slug: false
    updated: false
    comments: false
    link: false
    permalink: false
    excerpt: false
    categories: true
    tags: true
