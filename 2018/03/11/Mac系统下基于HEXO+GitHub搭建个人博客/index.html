1. 环境安装

在本地一共要安装 hexo/git/nodejs/homebrew 四个工具；

  • 安装Homebrew，它是一款Mac OS平台下的软件包管理工具，自行百度
  • 安装Git
    $ brew install git
  • 安装nodejs，直接去官网下载安装即可；
  • 安装hexo
    $ sudo npm install hexo-cli -g
